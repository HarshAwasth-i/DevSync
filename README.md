# DevSync 

A full-stack collaborative project management platform inspired by Jira and Trello.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MySQL
- JWT Authentication
- Socket.IO

## Features

- User Authentication
- Team Management
- Project Management
- Task Management
- Real-time Collaboration
- Notifications

## Status

🚧 Currently under development.
